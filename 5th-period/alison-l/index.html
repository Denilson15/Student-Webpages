    My Projects

    Simple Stopwatch

    This is just a simple stopwatch application that you can start, stop, and reset. It works by incrementing the digits on screen and updating each unit when the ticking time goes beyond that unit of time's bounds. It's great for timing some laps, cooking, or whatever else you need to time.

    To Do List

    Another simple program that allows users to add and subtract from their own custom to do list. It allows them to input their tasks and adds them to the list, and allows users to add tasks suggested by the application. Once a task is completed, it can be checked/crossed off on the list simply by pressing the task, or can be taken out of the list with the x button next to it.

    Connect Four Game

    A replica of the Connect 4 Game but the user can place the tokens in whichever cell as long as it is empty. There is a lot of manual, redundant code in this program which create both the board and the win conditions. If the user pressses an empty cell, it will update to contain their token and then check if that was the winning move(or if it will tie the game) based on the win conditions. Each time a token is placed, the tokens/players switch. If a win condition is met on either side or if the board is full and no one has won, the game will be end. Users can play again by clicking restart, clearing the board, and allowing them to play as many times as they'd like.
